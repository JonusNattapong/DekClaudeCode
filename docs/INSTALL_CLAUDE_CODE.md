# วิธีติดตั้งใช้งานใน Claude Code / Roo Code

## ขั้นตอนการติดตั้ง

1. **เปิดการตั้งค่า MCP Servers**
    - ใน Claude Code คลิกที่ Settings
    - เลือก `MCP Servers`
    - คลิกปุ่ม `Add Server`

2. **วางคอนฟิกนี้ลงไป**

    ```json
    {
      "mcpServers": {
        "thai-token-optimizer": {
          "command": "npx",
          "args": [
            "tsx",
            "d:/Projects/Github/DekClaudeCode/mcp-server.ts"
          ],
          "env": {
            "KILOCODE_API_KEY": "your_key_here"
          }
        }
      }
    }
    ```

3. **กด Save แล้ว Restart Claude**
    ตอนนี้ระบบจะทำงานอัตโนมัติทันที

## วิธีใช้งาน

**ไม่ต้องจำอะไรเลย!** พิมพ์ไทยธรรมดาตามปกติ ระบบจะแปลงให้อัตโนมัติเบื้องหลัง

| คุณพิมพ์ตามปกติ | ระบบแปลงเป็นอัตโนมัติ | ประหยัด |
|---|---|---|
| `ช่วยแก้บั๊กให้หน่อย` | `[FIX]` | 82% |
| `ช่วยเขียน unit test` | `[UT]` | 78% |
| `อธิบายโค้ดนี้ให้เข้าใจ` | `[EXPL]` | 75% |
| `เขียนใหม่ให้สะอาดกว่านี้` | `[REFACT]` | 71% |

## สถิติการใช้งาน (Statistics)

ระบบจะบันทึกการประหยัด Token ทุกครั้ง คุณสามารถดูสถิติได้โดยสั่ง:

```
แสดงสถิติการประหยัด
```

## การเพิ่มคำสั่งใหม่ (Expansion)

```
เพิ่มคำสั่ง "ช่วยตรวจสอบความปลอดภัย" เป็น "[SEC]"
```
